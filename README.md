# COM5001 Customer Churn Prediction
**Student:** Destiny Otto | **ID:** 22310489

This project predicts customer churn using Python and machine learning. We trained and compared Logistic Regression, Random Forest, and Support Vector Machine models on 10,000 bank customer records. The Random Forest model performed best with 86.2% accuracy and identified age, number of products, and activity level as the top reasons customers leave.

- `Churn_Modelling.csv` → Original dataset
